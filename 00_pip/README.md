## pip

pip é um gerenciador de pacotes Python.

##### Instalar pip
```
sudo apt-get install python-pip
```

##### Versão do pip
```
pip --version
```

##### Pacotes instalados
```
pip install --upgrade pip
```

##### Pacotes instalados
```
pip freeze
```

##### Instalar pacotes com pip
```
pip install <nome-pacote>
```

##### Desinstalar pacotes com pip
```
pip uninstall <nome-pacote>
```
