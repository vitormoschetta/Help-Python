## Algumas observações

- Os arquivos de teste precisam ter o prefixo 'test':
```
test_xpto.py
```

- As funções devem iniciar com o prefixo 'test':
```
def test_price_is_greater_than_zero():
    ...
```

- Cada pasta deve conter um arquivo init: 
```
__init__.py
```
